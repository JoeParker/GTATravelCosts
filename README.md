# GTA Travel Costs

Mod for the game Cyberpunk 2077 which adds a fare to the fast travel system. 
The cost scales with the distance travelled.

This mod is intended to be used alongside [GTA Travel](https://www.nexusmods.com/cyberpunk2077/mods/2006).

Settings can be tweaked in the native settings menu - [Mod Settings](https://www.nexusmods.com/cyberpunk2077/mods/4885) must be installed.
